# Top 10 Places Web Application

## Description
This is a simple, responsive web application that displays a list of the top 10 places to visit around the world. The application includes functionality to view predefined places and add new ones dynamically.

## Features
- Displays a grid of 10 popular travel destinations with images, names, and descriptions.
- Allows users to add new places through a form.
- Automatically updates the display upon adding new places.
- Styled with a clean, modern, responsive layout.
